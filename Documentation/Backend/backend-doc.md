# README
## Go to Backend Folder
```
cd backend
```
## Installation
```
npm install
```

## Running the app

#### Development
```
npm run start
```
#### Watch mode
```
npm run start:dev
```
#### Production mode
```
npm run start:prod
```

## Test

####  e2e Tests
```
npm run test:e2e
```
